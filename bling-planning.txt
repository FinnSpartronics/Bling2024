Modes:
    SOLID // Solid Primary Color
    SOLID_SECONDARY // Solid Secondary Color
    GRADIENT // Gradient from primary to secondary
    GRADIENT_REVERSED // Gradient from secondary to primary
    PULSE // Pulses primary color and off
    PULSE_SWITCH - Pulse between primary color and secondary color
    AROUND - Lights up one pixel to primary color at a time around the light strip to the primary color
    ALLIANCE_SOLID - Sets color to alliance
    ALLIANCE_PULSE - Alliance color pulses on and off
    ALLIANCE_PULSE_SWITCH - Alliance color pulses on and 
    ALLIANCE_AROUND - Lights up one pixel around the light strip to the alliance color

    OFF - No light
    WARNING - Flashes yellow and orange for warning
    ERROR - Flashes red and orange to show an error or warn of impending explosions

Colors: All default java colors

Changable brightness
